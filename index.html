Autonomous Chess Robot

 Our ME101 final project group found that online chess simply does not feel as good to play as physical chees. So, we built an automated physical chess robot.
 The mechanical portion was built upon the principles of a gantry system, using wheels, belts, racks and pinions, with all parts being custom modelled and 3D printed/laser cut or taken from a Tetrix/Lego EV3 kit. Uses a claw mechanism with custom gripper attachments to pick and move pieces
 The software was split into move calculation/detection and robot movement. The move calculation and detection is run on a laptop. This code is written in Python, using the Stockfish library for move calculation and OpenCV for computer vision and move detection. The robot itself runs on RobotC, taking a set of instructions from the laptop and parsing them into the correct chess movement. Also, including some other features like a resign button and screen displayed prompts.
